# NumberToWordConverter
NumberToWordConverter project is deisnged to convert numeral to English words. 
Example input and results:
1 - one
10 - ten
100 - one hundred
1000 - one thousand
Technologies
1. Maven - Build tool
2. Junit 4 - Unit Testing
3. Java 1.7 - Development
4. Eclipse - IDE
5. Git - Respository
